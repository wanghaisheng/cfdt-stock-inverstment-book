# 35.1 为什么不应该一开始编码？

原因：

你的 CFDT 最大价值不是数据。

而是：

你的交易认知。

---

比如：

你发现：

> 成交额跃迁 + 资金净流入 + 涨幅不大 = 可能机构吸筹。

---

这个规则：

不是程序员天然知道。

---

应该先：

让 AI 陪你观察100个案例。

---

流程：

```

↓

↓

AI分析

↓

人工确认

↓

形成规则

代码自动化
```

---

这才符合：

AI时代交易系统开发。

# 35.6 数据模型设计

不要先设计复杂指标。

先保存：

原始事实。

---

```typescript

code:string

name:string

industry:string

marketCap:number

}
```

---

## DailyMarket

```typescript
interface DailyMarket {

date:string

index:number

marketVolume:number

advance:number

decline:number

}
```

---

## StockDaily

```typescript
interface StockDaily {

code:string

date:string

turnover:number

}
```

---
