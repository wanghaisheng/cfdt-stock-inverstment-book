# 37.3 CFDT量能跃迁模型 V1

定义：

# Volume Jump Score

成交额跃迁评分。

---

## 指标1

成交额倍率

Volume Ratio

公式：

```
今日成交额 /
过去20日平均成交额
```

---

例如：

过去20日：

平均：

5亿。

---

今日：

15亿。

---

倍率：

3倍。

---

评分：

高。

---

代码：

TypeScript：

```typescript
function volumeRatio(
 today:number,
 average20:number
){
 return today / average20;
}
```

---

# 37.4 指标2

## 成交额加速度

Volume Acceleration

不是看一天。

看变化速度。

---

公式：

```
今日成交额增长率
-
昨日成交额增长率
```

---

例如：

成交额：

3亿

↓

5亿

↓

10亿

---

加速度明显。

---

说明：

资金关注增强。

---

---

TypeScript:

```typescript
function volumeAcceleration(
 today:number,
 yesterday:number,
 before:number
){

 const r1=(today-yesterday)/yesterday;

 const r2=(yesterday-before)/before;

 return r1-r2;

}
```

# 37.5 指标3

## 价格效率

Price Efficiency

这是CFDT核心。

---

问题：

资金进入后：

价格上涨多少？

---

公式：

```
价格涨幅 /
成交额增长
```

---

例如：

股票A：

成交额：

+200%

价格：

+5%

---

效率：

低。

---

可能：

吸筹。

---

---

股票B：

成交额：

+200%

价格：

+30%

---

效率：

高。

---

可能：

追涨或加速。

---

代码：

```typescript
function priceEfficiency(
 priceChange:number,
 volumeChange:number
){

 return priceChange / volumeChange;

}
```

# 37.6 指标4

## 资金持续性

Capital Persistence

---

单日资金：

容易骗人。

---

CFDT看：

连续性。

---

例如：

7日：

```
Day1 +3000万

Day2 +2000万

Day3 +5000万

Day4 +1000万
```

---

连续：

强。

---

代码：

```typescript
function flowPersistence(
 flows:number[]
){

const positiveDays =
flows.filter(
x=>x>0
).length;

return positiveDays / flows.length;

}
```
